CesiumExt 1 — JavaScript Toolkit for building world-class 3D globes and maps

CesiumExt is Open Source and enables building web applications with world-class 3D globes and maps. It is a JavaScript framework that is built based on CesiumJS and interface of the ExtJS library.
This library is inspired by the wonderful GeoExt library which was built on top of openlayers & ExtJS

CesiumExt 1 is the first base version which uses the versions CesiumJS 1.61 and ExtJS 6.2.0

About CesiumExt

This framework helps 3D GIS developers to design, develop, test applications in a rapid and more efficient way
